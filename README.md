**CrowdSense**

CrowdSense is a mini project developed as part of my academic curriculum. It is an AI and machine learning–based system designed to detect and count the number of people in a given scene using computer vision techniques. The goal of the project is to support crowd monitoring and management by providing real-time analysis and useful insights.

The system uses object detection models such as YOLO to identify people in images and video streams and estimate crowd density. It also includes basic trend-based analysis to observe how crowd patterns change over time, which can be helpful in areas like public safety, event management, and smart city applications.

Features
Real-time people detection and counting
Crowd density estimation
Basic trend analysis and prediction
Supports both image and video input
Technologies Used
Python
OpenCV
YOLO (You Only Look Once)
NumPy and Pandas for data processing
